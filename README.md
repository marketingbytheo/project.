# Studio — mia-obrt

Web stranica digitalnog dizajn studija. React + Vite + TypeScript + Tailwind CSS.

## Pokretanje

```sh
npm i
npm run dev
```

Otvori [http://localhost:8080](http://localhost:8080).

## Skripte

- `npm run dev` — razvojni poslužitelj
- `npm run build` — produkcijska build
- `npm run preview` — pregled builda
- `npm run lint` — ESLint

## Tehnologije

- Vite
- React 18
- TypeScript
- Tailwind CSS
- shadcn/ui (Radix)
- React Router
- Framer Motion
